关于 BitSapling

BitSapling 是一个使用 Java 语言基于 Spring Boot 的 BitTorrent Private Tracker。此程序旨在创建一个安全，高性能的 PT 程序， 以代替老旧的 NexusPHP。

尽管目前该项目很年轻，但我们正在逐步完善其功能。

关于更多 Sapling 的信息请查看仓库页面。

参与测试：

目前我们还没有完成用户系统和种子上传系统，因此您需要手动修改种子参与测试。

首先，在下载器中导入一个已经存在的种子，然后手动将其的 Tracker 服务器修改为 http://[域名]:8081/announce，请注意删除其他 Tracker，特别是 PT Tracker，否则您的 PT 站可能判定您作弊。
